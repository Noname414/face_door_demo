Functional requirements:

1. Face Registration

Create register_faces.py.

This script should:
- Load all images from known_faces/
- Treat each subfolder name as a person name
- Use InsightFace buffalo_l model
- Detect faces in each image
- If multiple faces are detected, use the largest face
- Extract normalized face embedding
- Average all embeddings for the same person
- Normalize the averaged embedding
- Save the face database to face_db.pkl

Expected command:
python register_faces.py

Expected output example:
[OK] professor: 1.jpg
[OK] professor: 2.jpg
[OK] student_A: 1.jpg
Face database saved to face_db.pkl
Total registered people: 3

2. Real-time Camera Recognition

Create recognize_camera.py.

This script should:
- Open webcam using OpenCV
- Load face_db.pkl
- Use InsightFace buffalo_l model
- Detect faces in real time
- Extract face embedding
- Compare current face embedding with registered embeddings using cosine similarity
- If similarity is higher than threshold, show the person name
- Otherwise show unknown
- Draw bounding box and label on camera frame
- Press q to exit

Expected command:
python recognize_camera.py

Display example:
professor 0.62
student_A 0.58
unknown 0.31

Recognition logic:
Use cosine similarity.

Pseudo logic:
similarity = dot(registered_embedding, current_embedding)

if best_similarity >= threshold:
    return person_name, best_similarity
else:
    return "unknown", best_similarity

Because InsightFace already provides normed_embedding, cosine similarity can be calculated with dot product.
